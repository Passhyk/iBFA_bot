##### PYTHON 3 PREREQUISITES
- pip3 
- selenium 
- urllib3 
### CLONING

```sh
$ git clone https://github.com/YurinDoctrine/iBFA_bot.git
```

### RUNNING

```sh
$ cd iBFA_bot/
$ sudo apt-get install python3-pip
$ pip3 install -r requirements.txt
$ bash start.sh
```
